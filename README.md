# 酒店管理系统 java765

## 项目概述

本项目是基于Spring Boot和Vue的前后端分离的酒店管理系统，适用于酒店与民宿管理，提供用户前台预订以及后台管理的全面解决方案。

## 技术栈

- 后端：Java，Spring Boot，MyBatis
- 前端：Vue，ElementUI
- 数据库：MySQL
- 构建工具：Maven，Node.js
- 开发环境：IntelliJ IDEA

## 功能模块

### 前台功能

- 用户注册登录
- 个人中心：信息修改、密码修改、我的订单
  - 订单状态：预约中、已支付
  - 订单评价：支付完成后可进行评价

#### 房间预订

- 全部房间

### 后台管理

1. 统计分析
   - 房间数量
   - 用户数量
   - 订单数量
   - 订单数量统计图
2. 会员管理
   - 查询
   - 删除
3. 广告管理
   - 新增
   - 修改
   - 删除
4. 分类管理
   - 查询
   - 新增
   - 修改
   - 删除
5. 房间管理
   - 查询
   - 新增
   - 修改
   - 删除
   - 修改预约状态
6. 订单管理
   - 查询
   - 删除
   - 查看订单状态
7. 评价管理
   - 查询
   - 删除
8. 管理员管理
   - 查询
   - 新增
   - 修改
   - 删除

## 系统角色

- 用户

## 核心亮点

- 前后端分离架构，便于维护与扩展
- 使用Spring Boot和Vue，提高开发效率
- MyBatis实现数据持久化，易于数据库管理
- 灵活的后台管理，满足各类管理需求

## 部署步骤

1. 环境搭建：安装JDK 8，MySQL 8.0，IntelliJ IDEA，Node.js 14，配置Maven
2. 数据库配置：导入SQL文件，创建数据库
3. 后端部署：编译后端代码，部署到服务器
4. 前端部署：编译前端代码，部署到Web服务器

请注意，这里不包含具体的部署细节，因为部署步骤可能因具体环境而异。
